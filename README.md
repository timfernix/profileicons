# Profileicons

A web application for exploring League of Legends summoner icons.

## What it does

This tool provides a centralized interface to browse, filter, and preview profile icons fetched directly from game data. It simplifies finding specific assets that are often hidden within the game client.

## Key Features

- **Live Search:** Instant filtering by icon name, ID, or description.
- **Deep Filtering:** Narrow down results by release year, rarity tiers (Standard to Transcendent), and legacy status.
- **Event Filtering:** Browse icons by their associated sets or events using an alphabetical navigation system.
- **Icon Previews:** View high-resolution assets and detailed metadata in an interactive modal.
- **Favorites:** Save specific icons to a persistent personal collection stored locally in the browser.
- **Sharing:** Generate direct links to specific icons for easy sharing.

## Data Source

The application dynamically fetches its data and images from [CommunityDragon](https://communitydragon.org/), ensuring it stays up-to-date with the latest League of Legends patches.
